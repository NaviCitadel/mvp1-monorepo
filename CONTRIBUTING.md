# How to clone this repo

## SSH setup

1. Create a password-protected SSH key on your PC;
2. Register your key on GitHub and setup your SSH config on your PC;

## Get started

1. Clone the repo:
   `git clone git@github.com:NaviCitadel/mvp1-monorepo.git`;
2. Write an Issue with your task name and write a good description and list of
   steps;
3. Create a branch named `dev-<issue-number>`, `fix-<issue-number>`, and push:
   `git switch -c <branch_name>`;
4. Create a commit and push it;
5. Create a pull request with some documentation in it. Wait for review.

## Sign commits and tags

This is optional, but if your want to tag, your will have to create a
password-protected GPG key and add it to your GitHub account. 
(see [documentation](https://docs.github.com/en/authentication/managing-commit-signature-verification/generating-a-new-gpg-key)).
